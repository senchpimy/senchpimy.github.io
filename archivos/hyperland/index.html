Hyperland

Para la configuracion de este use la configuracion ya existente de Hyde ya que esta contiene scripts que configuran y declaran ciertas variables en el sistema para tener la menor cantidad de problemas de compatibilidad con otros programas, lo que modifique en este caso fue el teclado y algunas configuraciones de la teclas:

Rofi es un programa muy simple que funciona tanto en xorg como en wayland solo hay que tener la version correcta instalada, para este use la configuracion por defecto que viene en hyde
