Sqrt

Este problema consisten en encontrar el numero entero m√°s cercano a la raiz de un numero dado.

Solucion

El problema consiste en una busqueda binaria en el rango desde 0 hasta el numero dado

func mySqrt(x int) int {
	st := 0
	max := x
	res := 0
	for st <= max {
		m := st + ((max - st) / 2)
		fmt.Println(m)
		sq := m * m
		if sq > x {
			max = m - 1
		} else if sq < x {
			st = m + 1
			res = m
		} else {
			return m
		}
	}
	return res
}
