Parentesis Valido

Este problema consisten en regresar cierto si un string contiene una serire de parentesis que sean validos

Solucion

def isValid(s: str) -> bool:
         fifo = []

         for char in s:
             if char in '({[':
                 fifo.append(char)
             else:
                 if not fifo:
                     return False

                 curr= fifo.pop()

                 if (char == ')' and curr!= '(') or (char == '}' and curr!= '{') or (char == ']' and curr!= '['):
                     return False

         return not fifo
